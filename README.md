# Reversi
A reversi game made with [vue](https://github.com/vuejs/vue) and [vue-cli](https://github.com/vuejs/vue-cli)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
